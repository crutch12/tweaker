This projects uses **pnpm**, so you should install it first: https://pnpm.io/installation

# Installation

```sh
$ pnpm i
```

# Development

```sh
$ pnpm dev
```

or

```sh
$ pnpm build:dev # 1st terminal
$ pnpm example:web # 2nd terminal
```

# Test

```sh
$ pnpm test
```

# E2E

```sh
$ pnpm e2e:ui # or pnpm e2e
```

# Build

```sh
$ pnpm build
```

# Release

```sh
$ pnpm release
```
